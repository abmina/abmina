## ⚡ Local-First AI. Zero Cloud.
I am the lead developer of **[Dark Matter IDE](https://github.com/abmina/dark-matter-ide)**. I gutted VS Code to create a privacy-first environment where your AI stack stays entirely on your hardware. 

- **Privacy:** Zero telemetry, zero data leaks.
- **Performance:** Optimized for local VRAM and high-speed indexing.
- **Freedom:** No subscriptions. Just your code and your model.
